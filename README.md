# antiamigo
Multiplayer browser game implemented using `Phaser 3` and `socket.io`

## Features
- Multiplayer (max 4 players)
- Kill others by jumping on their heads
- Scoring
- Different sprite for each player

## Prerequisites
- Node.js
- Express
- Nodemon
- Phaser 3
- socket.io 


## Usage
1. npm install
2. npm run
